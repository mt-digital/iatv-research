archive.org search results exporter

How to use

This tool allows you to export .csv versions of Internet Archive search results for custom analyses.

  1. Navigate to the Internet Archive's TV News Archive. Use the search interface to get a set of search results you want to export.
  2. When you have search results you want, copy the URL from your browser and paste it in the input box below.
  3. Press Go and a .csv-version download of the search results

Need help? Exported search results not what you expected? Contact Matt Turner, the site creator and administrator.
